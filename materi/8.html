SendKeys

Basic

  • untuk menulis keyboard secara otomatis sesuai perintah
CreateObject("wscript.shell").sendkeys ""

Example

set x = createobject("wscript.shell")

x.run "notepad.exe"

x.sendkeys "Hello there"

mengjeda sementara

  • 1000 : 1 detik
set x = createobject("wscript.shell")

x.run "notepad.exe"

wscript.sleep 3000
x.sendkeys "Hello there"

Enter

set x = createobject("wscript.shell")

x.run "notepad.exe"

wscript.sleep 3000
x.sendkeys "Hello there"
x.sendkeys "{enter}"
x.sendkeys "how are you doing"
